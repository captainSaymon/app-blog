# App Blog

Projekt na laboratoria z technologii aplikacji webowych.  
Projekt wykorzystuje **Angular CLI** i łączy się z przygotowanym wcześniej **[serwerem](https://github.com/captainSaymon/app-blog-server)**.


## Konfiguracja

Zainstalować zależności **(node modules)**:  
```bash
npm install
```

## Uruchomienie

Aby uruchomić lokalny serwer dewelopersk:

```bash
ng serve
```

Po uruchomieniu serwera, otwórz przeglądarkę i przejdź pod adres `http://localhost:4200`
